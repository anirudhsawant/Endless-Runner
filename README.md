INTRODUCTION: Cloud Integrated-Endless Runner Game  is an exhilarating endless runner game developed using the Unity game engine. Inspired by the popular game Subway Surfer, Urban Dash offers players a fast-paced and visually engaging experience as they navigate through a bustling urban environment. The game challenges players to dodge obstacles, collect coins, and complete missions while striving for high scores.
Gameplay and Features:
1.	Endless Running: The core gameplay involves continuous running through an ever-changing cityscape filled with obstacles such as vehicles, barriers, and construction sites. Players must swipe left, right, up, and down to dodge obstacles and keep running.
2.	Coin Collection: As players run, they collect coins scattered throughout the game environment. These coins can be used to purchase upgrades, power-ups, and new characters, enhancing the replayability of the game.
3.	Power-Ups: Urban Dash features various power-ups that provide temporary boosts to players. These include speed boosts, coin magnets, and invincibility, adding an extra layer of excitement and strategy to the gameplay.
4.	Character Customization: Urban Dash allows players to choose and customize their characters with different outfits and accessories. Unlockable characters each have unique abilities that can aid players in their runs.
5.	Stunning Graphics: Leveraging the power of the Unity game engine, Urban Dash features vibrant and detailed graphics. The dynamic urban environment is brought to life with smooth animations and a variety of visually appealing elements.
Development and Technology:
•	Endless Runner was developed using the Unity game engine, which provided a robust and flexible platform for creating high-quality 3D graphics and smooth gameplay mechanics. 
•	The game utilizes Unity’s physics engine to ensure realistic character movements and interactions with obstacles. 
•	Additionally, the game was designed with scalability in mind, ensuring optimal performance across various devices and screen sizes.
•	Endless Runner offers an exciting and immersive experience, combining fast-paced gameplay with visually stunning graphics and a wealth of features. 
•	With its engaging mechanics and social integration, the game provides endless entertainment for players of all ages.
